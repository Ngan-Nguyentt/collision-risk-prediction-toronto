# Collision Risk Predictive Analysis — Toronto

A machine-learning project predicting traffic collision risk in Toronto using
historical collision data, with feature engineering, model training, and
evaluation in a reproducible Jupyter notebook.

## Highlights
- Data cleaning and feature engineering on Toronto collision data
- Predictive modelling of collision risk
- Model evaluation and interpretation of results

## Tech
Python · pandas · scikit-learn · Jupyter

## Files
- `ML_Collision_Risk_Predictive_Analysis_in_Toronto.ipynb` — full ML notebook
